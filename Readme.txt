Program which found Button with id = 'make-everything-ok-button' in one document, and found this or similar button
in another document.
Input arguments: 1) Path for original document, where make-everything-ok-button will found
                 2) Path for document, where program should found similar button
    Not Required 3) Name of other element


Run application:
 java -jar AgileEngineTest.jar <path to original file> <path to second document>

Example:
 java -jar AgileEngineTest.jar D:\\test\sample-0-origin.html D:\\test\sample-1-evil-gemini.html

Program run on Java version: 11
